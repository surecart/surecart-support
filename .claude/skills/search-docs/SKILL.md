# Search Docs

Search SureCart documentation for user-facing information.

## When to use
When a support team member needs to find documentation about a SureCart feature, configuration, or how-to guide.

## Instructions

### Step 1: Understand the Query
Clarify what the user is looking for — a feature explanation, setup guide, configuration option, etc.

### Step 2: Search Multiple Sources
Search in this order:
1. **Wiki runbooks** — `.repos/surecart-support.wiki/` for troubleshooting guides and reference docs
2. **Official docs** — use the surecart-docs MCP server to query the documentation site
3. **Docs repo** — search `.repos/surecart-docs/` for documentation source files
4. **Plugin code** — search `.repos/surecart-wp/` for inline documentation and comments

### Step 3: Compile Answer
Provide:
- **Direct answer** — the information they're looking for, in plain language
- **Source** — where you found it (doc page name, file path)
- **Related docs** — other pages that might be helpful
- **Customer-facing link** — if there's a public doc page the support person can share with the customer
