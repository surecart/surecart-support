# Features

## What the Assistant Can Do

### 1. Answer Questions About SureCart
Ask anything about how SureCart works. The assistant searches:
- SureCart WordPress plugin source code
- SureCart platform (API) source code
- SureCart documentation
- Troubleshooting runbooks

### 2. Guided Troubleshooting (`/troubleshoot`)
Step-by-step diagnosis of customer issues:
- Classifies the issue type
- Searches relevant code and docs
- Provides resolution steps
- Suggests escalation if needed

### 3. API Documentation Lookup (`/lookup-api`)
Look up any SureCart API endpoint:
- Endpoint URL and HTTP method
- Required parameters
- Example responses
- Common errors

### 4. Documentation Search (`/search-docs`)
Search the official SureCart docs:
- Feature documentation
- How-to guides
- Configuration instructions

### 5. Code Flow Tracing (`/trace-flow`)
Understand how a feature works end-to-end:
- Entry points
- Processing steps
- Database interactions
- API calls

### 6. Error Code Lookup
Ask about any error code and get:
- What the error means
- Common causes
- Resolution steps

### 7. Auto-Updates
Every time you launch, the assistant automatically pulls:
- Latest plugin code
- Latest platform code
- Latest documentation
- Latest troubleshooting guides

## What the Assistant Cannot Do

- Modify any files or code
- Access live customer data (use the admin dashboard for that)
- Create GitHub issues or PRs
- Send messages to customers
- Make API calls to production systems
